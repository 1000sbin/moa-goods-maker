# 🍓 모아 굿즈메이커 (MOA Goods Maker)

투명 PNG를 올리면 **칼선·타공·고리·지지대**를 자동으로 만들어주는 굿즈 제작 도구예요.
디자인/칼선/화이트 레이어를 나눠서 **PSD·PDF·SVG·PNG**로 저장해요. 인쇄소에 그대로 넣을 수 있어요.

**설치 없이 브라우저에서 바로** — 파일 하나만 열면 끝. 이미지는 서버로 안 올라가요, 전부 내 컴퓨터에서 처리돼요.

## 4단계로 끝나요

### ① 시작 — 그림 올리고 크기 정하기
- 투명 PNG를 드래그해서 올려요 (JPG·흰 배경 이미지도 OK — ②에서 배경 제거 선택)
- **굿즈 타입** 고르기 (7종):
  | 타입 | 설명 |
  |---|---|
  | **스티커** | 외곽 칼선만 (구멍·고리 없음) |
  | **스티커 시트** | 여러 조각을 한 판에 |
  | **키링 · 구멍** | 그림 안쪽에 타공 구멍 |
  | **키링 · 고리** | 머리 위로 고리가 솟는 형태 |
  | **스탠드** | 촉 + 받침으로 세우기 |
  | **코롯토** | 굴러다니는 둥근 아크릴 |
  | **일반 아크릴** | 칼선만, 부속 없이 |
- **그림 실물 크기(mm)** 를 입력하면 해상도가 자동으로 맞춰져요
  - 클립스튜디오·포토샵 원고는 **DPI를 자동 감지**해요 (350dpi 원고면 자동으로 350)
- **대지 크기 지정**: 인쇄소가 "대지는 주문 사이즈와 동일하게" 요구하면 여기서 고정 (예: 50×50)
  - 대지를 켜면 미리보기에서 **그림을 드래그**해 위치를 옮길 수 있어요

### ② 칼선 — 모양 다듬기
- **여백(mm)**: 그림 바깥으로 얼마나 띄울지. 아크릴은 보통 2~3mm
- **부드럽게**: 칼선의 굴곡 정도 (각지게 ↔ 아주 뭉툭하게)
- **꼭지점 둥글리기(mm)**: 뾰족한 V홈을 둥글려요. 인쇄소가 "칼날이 못 들어간다"고 하면 0.5~1mm 주세요
- **틈 좁히기(mm)**: 머리카락 사이 같은 얇은 틈을 메워요
- **내부 구멍 메우기**: 팔·다리 사이 갇힌 틈의 안쪽 칼선을 없애요

### ③ 부속 — 고리·지지대
- **타공 지름 / 고리 벽 두께**: 열쇠고리가 걸리는 구멍과 그 주변 두께
- 미리보기에서 **고리·구멍·촉을 직접 드래그**해서 위치 조정 (인원별로 따로)
- **스탠드**: 촉 너비·높이, 받침 크기·모양(둥근/사각), 슬롯 위치
- **도형 추가**: 원·사각 등 기하 도형 칼선을 얹기

### ④ 저장 — 인쇄소에 넣기
- **화이트 레이어**: 아크릴 뒤에 깔 백판. 여백(mm)으로 죄거나 넓힐 수 있어요
  - **화이트 미리보기** 를 켜면 화면에서 백판 영역이 보여요
  - **화이트 형식**: 자동 / 래스터(이미지) / 벡터(패스)
- **PSD 색상 모드**: RGB / CMYK — 인쇄소가 CMYK 문서를 요구하면 선택
- **칼선 색상(PDF)**: RGB / CMYK 마젠타100 / 시안100 / 먹100 — "칼선은 M100으로" 요구에 대응
- 저장 형식: **PSD**(레이어 분리) · **PDF**(페이지 분리) · **SVG**(칼선 벡터) · **PNG 묶음 ZIP**
- **여러 장 한번에**: 같은 설정으로 이미지 여러 개를 한 번에 → ZIP

## 자주 묻는 것

**Q. 굿즈 실물 크기는 어디서 봐요?**
미리보기 아래 정보줄의 **"굿즈 실물"** — 여백·고리까지 포함된, 손에 쥘 아크릴의 진짜 크기예요. 발주서에 이 숫자를 쓰세요.

**Q. 클립스튜디오에서 만든 크기랑 달라요**
원고가 350dpi인데 도구가 300dpi로 읽으면 16.7% 차이 나요. ① 탭에서 **DPI 자동 감지** 표시를 확인하고, 안 맞으면 직접 입력하세요.

**Q. 고리가 완벽한 원이 아니에요**
정상이에요. 실제 고리가 걸리는 **타공 구멍은 정원**이고, 바깥 실루엣은 몸통과 이어지느라 살짝 물방울형이 돼요. 완전한 정원이면 목이 잘록해 부러지기 쉬워요.

**Q. 화이트가 그림보다 살짝 커요**
의도된 거예요. 인쇄 정합 오차를 흡수하려고 넉넉하게 덮어요. 반대로 작으면 조금만 틀어져도 가장자리에 흰끼가 생겨요. 신경 쓰이면 **화이트 여백을 −0.2mm** 정도 주세요.

**Q. 큰 굿즈(10cm 초과)를 만들어요**
화이트 **벡터 단순화가 자동으로 꺼져요** (큰 크기에선 단순화 오차가 눈에 띄어서). 20cm 이상이면 원고 자체를 더 큰 픽셀로 그리는 게 좋아요.

**Q. 슬라이더가 버벅여요**
꼭지점 둥글리기·틈 좁히기는 계산이 무거워요. 드래그 중엔 숫자만 바뀌고 손을 떼면 계산돼요. 같은 값으로 돌아오면 캐시가 있어 즉시 반영돼요.

**Q. 일러스트레이터로 넘기고 싶어요**
**PSD**(디자인·화이트 픽셀 + 칼선 벡터 마스크)를 열고, 재단선은 **SVG**를 따로 가져와 얹으면 제일 깔끔해요. CMYK 문서가 필요하면 ④에서 CMYK PSD로 받으세요.
